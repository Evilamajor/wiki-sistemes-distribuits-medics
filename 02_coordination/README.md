# 02 – Coordinació i consens

Aquesta secció recull els mecanismes que permeten als nodes d’un sistema distribuït
arribar a acords o mantenir una visió coherent de l’estat global.

Temes inclosos:
- Algorisme Raft
- Paxos
- Elecció de líder
- Gossip
- Reconciliació i protocols Bayou
- Consistència eventual vs forta
